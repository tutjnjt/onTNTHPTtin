Khái niệm AI

CHỦ ĐỀ 02 · Câu 5–9 (Phần I) + Câu 1 (Phần II Chung)
12B · MẠNG MÁY TÍNH & INTERNET
P.I: 2Biết 1Hiểu 1VD  |  P.II: aB bH c,dVD
🔌
Thiết bị mạng
  • Hub: phân phối tín hiệu TẤT CẢ cổng (cũ)
  • Switch: gửi đúng cổng đích theo MAC
  • Router: kết nối mạng khác nhau, định tuyến IP
  • Modem: chuyển đổi tín hiệu analog ↔ số
  • Access Point: phát sóng WiFi
📡
Giao thức TCP/IP
  • IP: địa chỉ và định tuyến gói tin
  • TCP: truyền dữ liệu tin cậy, kiểm lỗi
  • HTTP/HTTPS: truy cập web (HTTPS có mã hóa)
  • DNS: tên miền → địa chỉ IP
  • IPv4: 32 bit | IPv6: 128 bit
🖧
Chia sẻ tài nguyên
  • Chia sẻ thư mục qua Properties → Sharing
  • Phân quyền: Read / Read-Write
  • Truy cập: \\địa_chỉ_IP\tên_chia_sẻ
  • Workgroup: mạng ngang hàng nhỏ
  • Domain: mạng có máy chủ quản lý
🔢
Địa chỉ IP & DNS
  • IPv4: dạng a.b.c.d (VD: 192.168.1.1)
  • IP tĩnh: cấu hình thủ công, không đổi
  • IP động: cấp bởi DHCP server
  • DNS = "danh bạ điện thoại" của Internet
  • google.com → 142.250.190.14
🌐
CHỦ ĐỀ 03 · Câu 10–13 (Phần I) + Câu 5 (Phần II ICT)
12F · TẠO TRANG WEB (HTML & CSS)
P.I: 2Biết 1Hiểu 1VD  |  P.II ICT: a,bB cH dVD
🏗️
Cấu trúc HTML
  • <!DOCTYPE html> – khai báo loại tài liệu
  • <html> – thẻ gốc bọc toàn bộ
  • <head> – thông tin meta, title, CSS
  • <body> – nội dung hiển thị
  • <title> – tiêu đề hiển thị trên tab
📝
Thẻ HTML cơ bản
  • <h1>–<h6>: tiêu đề (h1 lớn nhất)
  • <p>: đoạn văn | <br>: xuống dòng
  • <b>/<strong>: in đậm
  • <a href="">: liên kết
  • <img src="" alt="">: hình ảnh
  • <ul><ol><li>: danh sách
📊
Bảng HTML
  • <table>: tạo bảng
  • <tr>: hàng (table row)
  • <th>: ô tiêu đề (đậm, giữa)
  • <td>: ô dữ liệu
  • colspan="2": gộp 2 cột ngang
  • rowspan="2": gộp 2 hàng dọc
🎨
CSS Định dạng
  • Selector tag: p { color: red; }
  • Selector class: .ten { font-size: 16px; }
  • Selector ID: #id { width: 100%; }
  • margin: ngoài viền | padding: trong viền
  • border: 1px solid #ccc;
  • Inline > Internal > External (ưu tiên)

CHỦ ĐỀ 04 · Câu 14–18 (Phần I)
10F · LẬP TRÌNH CƠ BẢN (PYTHON)
Câu 14,15: Biết   Câu 16,17: Hiểu   Câu 18: Vận dụng
📦
Biến & Kiểu dữ liệu
  • int: số nguyên (1, -5, 100)
  • float: số thực (3.14, -2.5)
  • str: chuỗi ("xin chào")
  • bool: True / False
  • list: danh sách [1, 2, 3]
  • print(), input() – nhập xuất
🔀
Lệnh rẽ nhánh
  • if điều_kiện: → lệnh
  • elif điều_kiện_2: → lệnh
  • else: → lệnh mặc định
  • So sánh: ==, !=, >, <, >=, <=
  • Logic: and, or, not
  • Lưu ý: Python dùng thụt lề (indent)
🔁
Vòng lặp
  • for i in range(n): – từ 0 đến n-1
  • for i in range(1, n+1): – từ 1 đến n
  • while điều_kiện: – lặp chưa biết số lần
  • break: thoát vòng lặp
  • continue: bỏ qua lần lặp hiện tại
🔧
Hàm (Function)
  • def ten_ham(tham_so):
  • return ket_qua
  • Gọi: bien = ten_ham(gia_tri)
  • Hàm có thể không có return
  • Biến cục bộ vs biến toàn cục
  • Đọc code: xác định input → output

CHỦ ĐỀ 07 · Câu 2 (Phần II Chung) + Câu 6 (Phần II ICT)
11F · CƠ SỞ DỮ LIỆU & SQL
P.II Chung: aB bH c,dVD  |  P.II ICT: a,bB cH dVD
📋
Khái niệm CSDL
  • Database: tập dữ liệu có cấu trúc
  • Table (Bảng): tập hợp dữ liệu cùng loại
  • Record (Bản ghi): một hàng trong bảng
  • Field (Trường): một cột trong bảng
  • Primary Key: khóa chính, định danh duy nhất
  • RDBMS: MySQL, Access, SQL Server
🔍
SQL – SELECT
  • SELECT * FROM Bảng;
  • SELECT cột1, cột2 FROM Bảng;
  • WHERE điều_kiện (=, >, <, LIKE)
  • ORDER BY cột ASC / DESC
  • LIKE "Nguyễn%" – tìm bắt đầu bằng
  • AND, OR, NOT – kết hợp điều kiện
📊
SQL – Hàm tổng hợp
  • COUNT(*): đếm số bản ghi
  • SUM(cột): tính tổng giá trị
  • AVG(cột): tính trung bình
  • MAX(cột): giá trị lớn nhất
  • MIN(cột): giá trị nhỏ nhất
  • GROUP BY: nhóm dữ liệu

Hình ảnh

Thẻ tự đóng, không cần </img>

Danh sách

ul: không thứ tự | ol: có số thứ tự
CSS màu sắc

Tên màu tiếng Anh hoặc mã hex
CSS hộp mô hình
margin: 10px; padding: 15px; border: 1px solid;
margin ngoài · padding trong · border viền

⚡ BẢNG TRA CỨU NHANH – CÂU LỆNH SQL TRỌNG TÂM

Lấy tất cả dữ liệu
SELECT * FROM HocSinh;
Lấy toàn bộ cột và hàng từ bảng HocSinh
Lọc có điều kiện
SELECT * FROM HocSinh WHERE DiemTB >= 8.0;
Lấy học sinh có điểm TB từ 8.0 trở lên
Sắp xếp kết quả
SELECT HoTen, DiemTB FROM HocSinh ORDER BY DiemTB DESC;
Sắp xếp giảm dần theo điểm TB
Đếm bản ghi
SELECT COUNT(*) FROM HocSinh WHERE Lop = '12A1';
Đếm số học sinh lớp 12A1
Tính trung bình
SELECT AVG(DiemTB) FROM HocSinh;
Tính điểm trung bình của tất cả học sinh
Tìm kiếm chuỗi
SELECT * FROM HocSinh WHERE HoTen LIKE 'Nguyễn%';
% thay cho nhiều ký tự bất kỳ
